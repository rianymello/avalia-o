CONHECIMENTO

"Um leitor vive mil vidas antes de morrer. O homem que nunca lê vive apenas uma."

- George R.R. Martin

Escolhi o tema da livraria porque a leitura sempre foi uma paixão profunda na minha vida. Através dos livros, posso explorar novos mundos, adquirir conhecimento e viver inúmeras aventuras. É com esse espírito de descoberta e fascinação que convido todos a se perderem entre as páginas de nossa coleção.

Eu usei os botões para criar transições entre as páginas. Ao adicionar uma ação a um botão, podemos definir o comportamento desejado quando o usuário clica nele. Por exemplo, ao clicar em um botão "Próximo", a aplicação pode navegar para a próxima página ou exibir um novo conteúdo.


Botões

As labels foram utilizadas para exibir textos estáticos, como títulos, descrições ou instruções aos usuários. Elas são úteis para proporcionar um contexto ou informações adicionais sobre os elementos da interface.

As picture boxes foram empregadas para exibir imagens dentro do formulário. Isso inclui ícones, fotos ou qualquer gráfico que necessite ser mostrado ao usuário. As picture boxes são configuráveis em termos de tamanho, posição e imagem carregada, permitindo flexibilidade no design da interface.

Para este projeto, utilizei a linguagem C#. C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft como parte da sua plataforma .NET. É amplamente usada para desenvolver aplicativos desktop, web e serviços web devido à sua simplicidade e poder expressivo.


C#

Usando C# no Visual Studio, fui capaz de criar formulários interativos com uma interface de usuário rica, implementando funcionalidades específicas como transições de páginas através de botões, exibição de textos com labels e apresentação de imagens


Projeto Final

Essa é a "Home" do programa, ela possui todos os livros com imagem das capas e um botão para cada um que leva a página correspondente ao livro aonde o utilizador pode encontrar mais informações sobre cada um.

Nessa pagina eu usei 2 TextBox, um para senha que é criptografado com "*" e o outro para o nome de utilizador que é salvo para ser usado no seja bem vindo da proxima pagina

Na primeira página de livros eu coloquei uma imagem do livro correspondente ao que o utilizador clicar e também as informações de nome do livro, gênero, editora, e autor.

Quando o utilizador clica em editar ele é levado a outra página de livros que tem TextBox que podem editar as informações atuais do Livro.
